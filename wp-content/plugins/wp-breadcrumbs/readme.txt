== Description ==

Outputting breadcrumbs when using the function wp_breadcrumbs()

== Installation ==

1. Upload `wp_breadcrumbs.php` to `/wp-content/plugins/wp-breadcrumbs/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the function wp_breadcrumbs() in a page template.
